# Fake-Virus.js
Simple Fake Virus using JavaScript. 
The setup is straightforward. 
We will set it up as a bookmarklet using the Blooby Boi bookmarklet runner.

Step 1: is to go to this website https://blobby-boi.github.io/BlobeBM/main.html.

Step 2: Click Add bookmark and copy the code for the "Fake Virus.js" section.

Step 3:Once you add the bookmark, name it anything you feel like. Then click the Blue button, and it should run.

If the website is blocked and you are at school, go to this link: https://astroid.wtf.cdn.cloudflare.net
Then click anything on the page. Once you are on the page, click "Extras" then click "Launch about:blank"
Then go to the globe that says "Search" and copy the link from step 1, and follow the rest of the steps.
